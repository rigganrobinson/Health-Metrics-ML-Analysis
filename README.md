# Health-Metrics-ML-Analysis
End-to-end medical data analysis and diabetes risk prediction project using Python, Pandas, and Scikit-learn. Includes data cleaning, EDA, and Random Forest classification.
🩺 Health Metrics & Diabetes Risk Prediction
This project focuses on analyzing lifestyle and biological health metrics to predict diabetes risk and visualize the relationships between different health factors.

🚀 Project Overview
By analyzing critical metrics such as glucose levels, blood pressure, BMI, and age, this project aims to:

Data Cleaning: Handle missing or biologically impossible values (e.g., zero-values in glucose).

EDA (Exploratory Data Analysis): Discover correlations between lifestyle habits and health risks.

Machine Learning: Predict diabetes risk using a classification model.

🛠 Tech Stack
Python 3.13

Pandas: Data manipulation and cleaning.

Seaborn & Matplotlib: Static data visualization.

Plotly: Interactive data dashboards.

Scikit-Learn: Machine Learning (Random Forest Classifier).

📊 Key Insights
Data Preprocessing: Biologically impossible '0' values in Glucose and Blood Pressure columns were replaced with median values to maintain data integrity.

Risk Correlation: Analysis reveals a significant increase in risk factors for individuals with a BMI over 30.

Age Factor: A strong positive correlation was observed between age and cholesterol levels.
